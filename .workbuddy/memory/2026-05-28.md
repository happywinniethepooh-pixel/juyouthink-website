## Vercel 部署三大问题诊断与修复

### 问题1：中英文切换在 Vercel 失效
- **根因**：team.html 等子页面 `.lang-zh` 元素无内联 `display:none`，页面加载时双语同时显示；lang.js 执行时机在 Vercel 上不稳定
- **修复**：重写 lang.js，在 `<head>` 注入 `.lang-zh { display:none !important }` 的基线 CSS，确保页面加载时立即隐藏中文；setLang() 调用时删除该规则并重新控制；加入 DOMContentLoaded 保护

### 问题2：图片在 Vercel 不显示
- **根因**：Vercel（Linux）文件系统大小写敏感；本地文件路径正确（images/juyou-logo-blue.png、images/logo_*.png、images/pptx-extracted/image9/10.png）
- **真正根因**：图片文件没有被 git commit（在 git 仓库建立后才生成的 logo_*.png 透明处理文件）
- **修复方案**：需要用户手动 git add images/ && git commit && git push

### 问题3：team 页面字体太大
- **根因**：.team-name 1.1rem + 双列布局在小屏宽不够
- **修复**：.team-name 缩小至 1rem，加 text-overflow:ellipsis；.team-title 缩小至 0.72rem
