# JuYou Think 网站部署指南

> **网站文件**：7 个文件（`index.html` + 5 个子页面 + `shared.css`）
> **域名**：`www.juyouthink.com`
>
> **文件清单**：
> - `index.html` — 首页
> - `services.html` — 服务介绍页
> - `why-us.html` — 核心优势页
> - `cases.html` — 案例展示页
> - `team.html` — 团队介绍页
> - `contact.html` — 联系我们页
> - `shared.css` — 共享样式文件

---

## 一、整体思路

这是一个纯静态多页面网站，不需要服务器、不需要数据库，部署方式非常简单。整个流程分三步：

1. 选择一个**静态托管平台**上传文件
2. 在托管平台绑定你的**自定义域名**
3. 在**域名注册商**那里配置 DNS 解析，让域名指向托管平台

> **重要**：上传时需要将所有 7 个文件放在**同一个文件夹**内，确保 `shared.css` 与各 HTML 文件同级。

## 二、推荐方案对比

| 方案 | 推荐指数 | 特点 | 费用 |
|------|---------|------|------|

## 三、方案 A：Netlify 部署（最简单，推荐入门）

### 第一步：注册 Netlify

1. 打开 [https://www.netlify.com](https://www.netlify.com)
2. 点击 **Sign up**，用 Google 或 GitHub 账号注册（或直接邮箱注册）
3. 登录后进入控制台

### 第二步：上传网站文件

方法 1（最简单 — 拖拽）：
1. 在控制台首页，找到 **"Sites"** 标签页
2. 页面底部有一个虚线框，提示 **"Drag and drop your site output folder here"**
3. 将包含所有文件的**文件夹**直接拖进去（确保 `index.html`、`shared.css` 和其他 HTML 文件都在同一层级）
4. 等待几秒，Netlify 自动完成部署，给你分配一个临时域名（如 `quirky-babbage-123.netlify.app`）

### 第三步：绑定自定义域名

1. 在网站控制台，点击 **"Domain settings"** → **"Add custom domain"**
2. 输入 `www.juyouthink.com`，点击确认
3. Netlify 会显示需要配置的 DNS 记录，复制以下信息备用：
   - 类型：`CNAME`
   - 名称：`www`
   - 值：类似 `yoursite.netlify.app`（或 Netlify 给你的具体值）

## 四、方案 B：Vercel 部署（速度最快，推荐长期使用）

### 第一步：注册 Vercel

1. 打开 [https://vercel.com](https://vercel.com)
2. 用 GitHub 账号注册（需要先有 GitHub 账号）

### 第二步：将文件上传到 GitHub

1. 打开 [https://github.com](https://github.com)，注册/登录
2. 点击右上角 **"+"** → **"New repository"**
3. 仓库名填 `juyouthink-website`，勾选 **Public**，点击 **Create repository**
4. 在仓库页面，点击 **"Add file"** → **"Upload files"**
5. 将所有 7 个网站文件一起上传（`index.html`、`services.html`、`why-us.html`、`cases.html`、`team.html`、`contact.html`、`shared.css`），点击 **Commit changes**

### 第三步：在 Vercel 导入项目

1. 登录 Vercel 后，点击 **"Add New Project"**
2. 选择刚才创建的 GitHub 仓库 `juyouthink-website`
3. 保持默认设置，点击 **"Deploy"**
4. 等待 1 分钟，部署完成，获得临时域名（如 `juyouthink-website.vercel.app`）

### 第四步：绑定自定义域名

1. 进入项目设置 → **"Domains"**
2. 输入 `www.juyouthink.com`，点击 **Add**
3. Vercel 会显示需要配置的 DNS 信息：
   - 类型：`CNAME`
   - 名称：`www`
   - 值：`cname.vercel-dns.com`

## 五、配置域名 DNS 解析（关键步骤）

> 无论选择哪个托管平台，这一步都是必须的。你需要登录**购买 www.juyouthink.com 域名的注册商网站**。

### 如何找到域名注册商？

- 如果域名在**阿里云（万网）**购买：登录 [aliyun.com](https://www.aliyun.com) → 控制台 → 域名
- 如果在**腾讯云**购买：登录 [cloud.tencent.com](https://cloud.tencent.com) → 域名
- 如果在 **GoDaddy** 购买：登录 [godaddy.com](https://www.godaddy.com)
- 如果在 **Namecheap** 购买：登录 [namecheap.com](https://www.namecheap.com)

### DNS 配置步骤（以阿里云为例）

1. 登录阿里云控制台 → **域名** → 找到 `juyouthink.com`
2. 点击 **"解析"** 或 **"管理 DNS"**
3. 点击 **"添加记录"**，填写以下信息：

**主域名配置**（www）：
| 记录类型 | 主机记录 | 记录值 |
|---------|---------|--------|
| CNAME | `www` | 托管平台给你的值（如 `cname.vercel-dns.com`） |

**根域名配置**（可选，让 juyouthink.com 也能访问）：
| 记录类型 | 主机记录 | 记录值 |
|---------|---------|--------|
| CNAME | `@` | 托管平台给你的根域名值 |

4. 点击确认，等待 **5-30 分钟** DNS 生效

### 验证 DNS 是否生效

打开命令提示符（Windows 按 Win+R，输入 cmd），执行：
```
nslookup www.juyouthink.com
```
如果返回的地址指向托管平台，说明配置成功。

## 六、启用 HTTPS（自动完成）

- **Netlify 和 Vercel**：绑定域名后，HTTPS 证书**自动申请、自动续期**，无需任何操作
- 证书通常在 DNS 生效后 **5-10 分钟内** 自动完成
- 完成后访问 `https://www.juyouthink.com` 即可看到网站，浏览器地址栏显示🔒锁标

## 七、后续更新网站内容

### Netlify 更新方法
1. 修改本地的 HTML 或 CSS 文件
2. 登录 Netlify → 进入网站 → **"Deploys"** → 拖入更新后的文件夹（包含所有文件）
3. 等待 1-2 分钟自动更新

### Vercel 更新方法
1. 修改本地的 HTML 或 CSS 文件
2. 上传到 GitHub 仓库（替换原文件）
3. Vercel **自动检测更新并重新部署**（全程无需手动操作）

## 八、常见问题

**Q：DNS 配置后多久生效？**
通常 5-30 分钟，最长 48 小时（罕见）。

**Q：域名显示不安全？**
稍等几分钟，HTTPS 证书需要时间颁发。也可以在托管平台控制台手动触发证书申请。

**Q：根域名 juyouthink.com（不带 www）能访问吗？**
Netlify 和 Vercel 都支持同时绑定根域名，建议也添加 `juyouthink.com`（不带 www），并设置**自动重定向**到 www 版本。

**Q：网站加载慢怎么办？**
Vercel 和 Netlify 都有全球 CDN，对海外用户来说速度已经很快。如果主要服务海外客户，这两个平台完全足够。
